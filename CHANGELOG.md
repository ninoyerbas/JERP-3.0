## [3.0.0] - 2026-02-06

### Initial Release

**Author:** Julio Cesar Mendez Tobar

#### Features
- ✅ Complete Financial Management (GL, AP, AR)
- ✅ Chart of Accounts with FASB/ASC support
- ✅ Journal Entries with posting system
- ✅ Bills and Invoices management
- ✅ Customer and Vendor management
- ✅ Human Resources module
- ✅ Employee management
- ✅ Timesheet tracking
- ✅ Payroll processing
- ✅ Compliance and reporting
- ✅ Multi-tenant architecture
- ✅ WPF Desktop application
- ✅ RESTful API backend
- ✅ SQL Server database

#### Architecture
- Clean Architecture pattern
- MVVM for WPF UI
- Entity Framework Core
- Dependency Injection
- Multi-tenant data isolation

#### Cannabis Industry Features
- METRC API integration
- 280E tax compliance
- Inventory tracking with batch/lot management
- Point of Sale system
- Age verification
- State reporting
